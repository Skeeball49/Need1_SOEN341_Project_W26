# MealMajor (SOEN 341 — Team Project W26)

MealMajor is a web application designed for students to plan meals, manage groceries, and build easy, healthy recipes. The goal is to help students save time, stay consistent with healthy eating, and organize their weekly meals in a simple and practical way. 

## Project Objective
This project follows an Agile/Scrum approach across multiple sprints. We use GitHub for version control, issue tracking (user stories + tasks), and project documentation. The end result is expected to be at least a compelling middle-fidelity prototype. 
## Baseline Features (Project Scope)
- **User Account Management**
  - Registration and login
  - Profile management (diet preferences, allergies)
- **Recipe Management**
  - Create/edit/delete recipes (ingredients, prep time, steps, cost, etc.)
  - Search and filter recipes (time, difficulty, cost, dietary tags, etc.)
- **Weekly Meal Planner**
  - Weekly grid view (days + meal types)
  - Assign recipes to meals
  - Edit/remove meals and prevent duplicates in the same week 

> By mid-semester, the team will propose and implement an additional feature integrated with the baseline features.

## How to Run (Prototype)Clone the repository to your local machine.
- Navigate to the project root directory.
- Initiate the project using npm init -y
- Install the required dependencies using `npm install`that will install the modules
- Start the application by running `node SOEN341_Project/index.js` (or `npm run dev` if nodemon is installed).
- Open a web browser and access the application at `http://localhost:3000`.

This prototype demonstrates basic user registration, login, and profile display using a Node.js, Express, and EJS-based frontend.
